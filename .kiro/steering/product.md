# Product Overview

This is a Vue 3 + TypeScript Admin UI template designed for building enterprise administrative applications. The template provides a production-ready foundation with OAuth 2.0 authentication (Entra ID), role-based access control (RBAC), user management, and activity tracking capabilities.

## Key Features

- OAuth 2.0 authentication with PKCE flow
- Role-based access control with permission-based UI rendering
- User management interface with role assignment
- Activity tracking and audit logs for compliance
- Responsive layout with collapsible sidebar
- Light/dark theme support
- Global notification system
- Breadcrumb navigation

## Target Use Cases

Administrative dashboards and internal tools requiring enterprise identity management, user access control, and audit logging capabilities.
